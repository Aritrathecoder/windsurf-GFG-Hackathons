# WindSurf - AI-Powered BI Dashboard

Intelligent Business Intelligence (BI) system that allows non-technical users to generate fully functional, interactive data dashboards using natural language prompts.

## Features
- **Natural Language Parsing**: Uses LLMs to translate regular English queries into SQL and chart data.
- **Dynamic Dashboards**: Real-time rendering of metrics, insights, and interactive charts.
- **Performance Optimized**: Built with Next.js and Tailwind CSS for speed and responsiveness.
- **Comprehensive Analytics**: Supports Bar, Line, Pie, and Area charts with automatic data-key detection.

## Tech Stack
- **Framework**: Next.js (App Router)
- **AI**: Groq API (Llama 3.3 70B)
- **Database**: SQLite
- **Styling**: Tailwind CSS & Glassmorphism UI
- **Visuals**: Recharts for interactive data visualization

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Configure your API keys (GROQ_API_KEY) in `.env`
4. Run the development server: `npm run dev`
